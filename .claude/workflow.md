# Workflows

## Full Feature Development Workflow

### Phase 1: Planning (Product Manager)
- Read product-manager.md
- Verify feature aligns with priorities
- Define acceptance criteria

### Phase 2: Implementation (Senior Engineer)
- Read senior-engineer.md
- Read relevant existing code
- Implement with tests
- Self-review against standards

### Phase 3: Review (Lead Engineer)
- Read lead-engineer.md
- Review all changes
- Check against product requirements
- Approve or request changes

### Phase 4: Documentation (Tech Writer)
- Read tech-writer.md
- Document new APIs/features
- Update relevant docs
- Keep it minimal but complete
